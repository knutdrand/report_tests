============
Installation
============


Stable release
--------------

To install Report Tests, run this command in your terminal:

.. code-block:: console

    $ pip install report_tests

or for the newest:

.. code-block:: console

    $ pip install git://github.com/knutdrand/report_tests


If you don't have `pip`_ installed, this `Python installation guide`_ can guide
you through the process.

.. _pip: https://pip.pypa.io
.. _Python installation guide: http://docs.python-guide.org/en/latest/starting/installation/


From sources
------------

The sources for Report Tests can be downloaded from the `Github repo`_.

You can either clone the public repository:

.. code-block:: console

    $ git clone git://github.com/knutdrand/report_tests

Once you have a copy of the source, you can install it with:

.. code-block:: console

    $ pip install -e .


.. _Github repo: https://github.com/knutdrand/report_tests
.. _tarball: https://github.com/knutdrand/report_tests/tarball/master
