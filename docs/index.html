Installation¶

Stable release¶

To install Report Tests, run this command in your terminal:

$ pip install report_tests

or for the newest:

$ pip install git://github.com/knutdrand/report_tests

If you don’t have pip installed, this Python installation guide can guide you through the process.

From sources¶

The sources for Report Tests can be downloaded from the Github repo.

You can either clone the public repository:

$ git clone git://github.com/knutdrand/report_tests

Once you have a copy of the source, you can install it with:

$ pip install -e .

Usage¶

To use Report Tests in a project:

import report_tests
